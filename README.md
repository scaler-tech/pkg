# scaler-tech/pkg

Homebrew tap and Scoop bucket for [scaler-tech](https://github.com/scaler-tech) projects.

Manifests are published automatically by [GoReleaser](https://goreleaser.com) on each release.

## Install toad

### macOS and Linux (Homebrew)

```console
brew tap scaler-tech/pkg https://github.com/scaler-tech/pkg
brew install --cask toad
```

### Windows (Scoop)

```console
scoop bucket add scaler-tech https://github.com/scaler-tech/pkg
scoop install toad
```
